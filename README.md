# MineSweeper
Just for fun implementation of MineSweeper with the option to let the Computer solve it

Commands:     - left click      : open one cell
              - right click     : flag a cell as bomb
              - Escape          : close game
              - n               : solve next Step
              - a               : automode to solve all
