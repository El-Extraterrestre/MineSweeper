# MineSweeper
Just for fun implementation of MineSweeper with the option to let the Computer solve it

Commands:     - left click      : open one cell\n
              - right click     : flag a cell as bomb\n
              - Escape          : close game\n
              - n               : solve next Step\n
              - a               : automode to solve all\n
